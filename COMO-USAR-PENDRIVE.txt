🚀 SISTEMA PNSB - USO EM PENDRIVE
=====================================

📋 PRÉ-REQUISITOS:
- Windows 10/11
- Node.js instalado (https://nodejs.org/)
- Navegador web

🎯 OPÇÃO 1: USAR DIRETAMENTE DO PENDRIVE

1️⃣ INICIAR SISTEMA:
   - Duplo-clique em: iniciar-sistema.bat
   - Aguarde a inicialização automática
   - O navegador abrirá automaticamente

2️⃣ USAR O SISTEMA:
   - Login: Gustavo (usuário comum)
   - Login: admin (administrador)
   - Use normalmente

3️⃣ PARAR SISTEMA:
   - Duplo-clique em: parar-sistema.bat
   - Ou feche as janelas do terminal

🎯 OPÇÃO 2: COPIAR PARA COMPUTADOR (RECOMENDADO)

1️⃣ COPIAR SISTEMA:
   - Duplo-clique em: copiar-para-computador.bat
   - Digite o caminho (ex: C:\SistemaPNSB)
   - Aguarde a cópia ser concluída

2️⃣ USAR SISTEMA COPIADO:
   - Navegue até a pasta copiada
   - Duplo-clique em: iniciar-sistema.bat
   - Use normalmente

✅ VANTAGENS DE COPIAR PARA PC:
- Mais rápido (não depende do pendrive)
- Mais estável (sem problemas USB)
- Mais seguro (dados no HD)
- Atalho na área de trabalho
- Backup automático
